# Docker login
```
docker login
```

# Building the docker

```
docker build . -t prashantdey/node-mongo-world
```

# Look for the build image

```
docker images
```

# Run

```
docker-compose up
```

If you want to stop the container then run the command:
```
docker-compose down
```

# Get container ID

```
docker ps
```

# Print app output

```
docker logs <container id>
```
